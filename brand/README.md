# Brand Assets

Source files for the RutOS integration icon and logo.

## Home Assistant Integration Icon

To display the icon in the Home Assistant UI, submit a PR to the
[home-assistant/brands](https://github.com/home-assistant/brands) repository
with PNG exports under `custom_integrations/rutos/`:

| File          | Size         |
| ------------- | ------------ |
| `icon.png`    | 256 x 256 px |
| `icon@2x.png` | 512 x 512 px |

Export from `icon.svg` at both sizes. PNGs must have transparent backgrounds and
be losslessly compressed.
